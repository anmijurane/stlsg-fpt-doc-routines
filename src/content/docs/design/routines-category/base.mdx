Este diseño corresponde a la propuesta original y se utilizó como base para desarrollar la propuesta visual. No se realizaron modificaciones visuales, salvo algunos ajustes mínimos de detalle.

### Colapsado

Esta pantalla tiene como propósito principal presentar al usuario las diferentes categorías
de rutinas disponibles dentro de la aplicación del gimnasio Planet Fitness.
Sirve como punto de entrada para explorar ejercicios según el objetivo del usuario:
adaptación, salud, ganancia muscular o quema de grasa.

![collapsed](./assets/base_collapsed.png)

### Expandido

Permitir al usuario explorar de forma visual e intuitiva las rutinas disponibles
dentro de la categoría expandida, presentadas por niveles de dificultad.
Cada tarjeta ofrece información esencial para que el usuario seleccione la rutina
que más se adecúe a su nivel y necesidades.

![expanded](./assets/base_expand.png)

### Scoll

Al hacer scroll, el sistema colapsa automáticamente el encabezado, reacomodando el ícono
y el título para mantener una referencia visual compacta. Este comportamiento está diseñado
para optimizar el espacio en pantalla, facilitando la lectura de los contenidos sin invadir
ni saturar la experiencia del usuario. La estructura mantiene la continuidad del contexto
actual, permitiendo navegar entre niveles sin perder de vista la categoría activa.

Este enfoque mejora significativamente la usabilidad en dispositivos móviles,
al priorizar la visibilidad de los elementos interactivos (tarjetas de rutina)
y reducir la interferencia visual del header original.

![scrollling](./assets/base_scrollling.png)
