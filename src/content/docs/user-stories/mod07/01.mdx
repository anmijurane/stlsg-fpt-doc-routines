## Historia de usuario:

**dado** que ya seleccione un ejercicio del listado de _"los ejercicios en la rutina"_

**como** usuario/socio de Planet Fitness,

**quiero** poder marcar un ejercicio como realizado desde la vista de rutina o desde el detalle del ejercicio, para llevar un control visual de mi progreso durante la sesión.

## Criterios de aceptación:

  - El usuario puede marcar como realizado un ejercicio desde la pantalla de rutina.
  - El usuario puede marcar como realizado un ejercicio desde la pantalla de detalle.
  - Al marcar como hecho, debe reflejarse un cambio visual claro.
  - El estado debe persistir en la sesión actual.
