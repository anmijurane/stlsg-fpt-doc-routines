## Historia de usuario:

**dado** que ya estoy en el sitio, y se abrio el modal

**como** usuario/socio de Planet Fitness,

**quiero** que el sistema registre cada vez que un usuario entra a una página o rutina, incluyendo la hora y sección visitada,

**para** analizar el comportamiento de navegación dentro de la plataforma.

## Criterios de aceptación:

  - Cada vez que se carga una vista o sección, se debe registrar:
  - Fecha completa (día, mes, año)
  - Hora exacta (HH:MM:SS)
  - Nombre de la sección o rutina visitada
  - Este evento debe enviarse al backend o almacenarse en la base de datos.
