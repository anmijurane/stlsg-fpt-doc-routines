## Historia de usuario:

**como** sitio,

**quiero** saber cuáles son las secciones o rutinas más visitadas,

**para** tomar decisiones sobre qué contenidos mantener, mejorar o destacar.

## Criterios de aceptación:

  - El sistema debe contar el número total de accesos por cada sección o rutina.
  - La información debe estar organizada por fecha y hora para facilitar análisis cronológico.
  - Los datos deben estar disponibles para su uso en reportes o dashboards.
