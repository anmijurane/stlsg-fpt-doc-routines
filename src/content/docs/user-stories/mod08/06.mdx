## Historia de usuario:

**dado** que ya estoy en el sitio, y se abrio el modal

**como** usuario/socio de Planet Fitness,

**quiero** conocer cuánto tiempo permanece un usuario en cada pantalla o sección,

**para** entender qué contenidos generan mayor interés o retención.

## Criterios de aceptación:

  - El sistema debe comenzar a contar el tiempo al cargar una pantalla.
  - El tiempo debe detenerse y registrarse cuando el usuario:
    - Cambia de sección
    - Cierra la pestaña
    - Recarga la página
  - El tiempo debe almacenarse junto con la fecha, hora y nombre de la sección.
