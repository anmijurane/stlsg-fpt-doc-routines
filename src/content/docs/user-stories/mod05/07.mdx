## Historia de usuario:

**dado** que ya seleccione un ejercicio del listado de _"los ejercicios en la rutina"_

**como** usuario/socio de Planet Fitness,

**quiero** ver una ilustración que indique qué músculo estoy trabajando con el ejercicio,

**para** tener mayor conciencia corporal y planificar mejor mi entrenamiento.

## Criterios de aceptación:

  - Debe mostrarse una imagen o esquema del cuerpo con el músculo trabajado resaltado.
  - La imagen debe ser coherente con el tipo de ejercicio y su efecto físico.
  - Debe colocarse cerca de los parámetros del ejercicio, manteniendo el orden visual.
  - El músculo resaltado debe estar marcado con un color visible (ej. azul).
