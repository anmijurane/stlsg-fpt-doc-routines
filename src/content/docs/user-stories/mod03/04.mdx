**dado** que ya seleccione una rutina del listado de _"Exploración de rutinas"_

**como** usuario/socio de Planet Fitness,

**quiero** regresar a la pantalla de _Exploración de rutinas_,

**para** seleccionar una rutina diferente en caso de error o cambio de preferencia.

## Criterios de aceptación:

- Cuando oprima el botón debo regresar a la pantalla de _Exploración de rutinas_.
- Se debe mostrar cerradas las secciones de las categorias.
-	El botón debe decir “Regresar”.
-	Al hacer clic, debe volver a la pantalla de selección de categoría y nivel.
-	No se debe perder la sesión ni el identificador del club.
