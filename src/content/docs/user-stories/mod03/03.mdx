**dado** que ya seleccione una rutina del listado de _"Exploración de rutinas"_

**como** usuario/socio de Planet Fitness,

**quiero** tener un botón para iniciar la rutina después de leer la información,

**para** comenzar con los ejercicios del nivel seleccionado.

## Criterios de aceptación:

	-	El botón debe decir “Continuar”.
	-	Al hacer clic, debe navegar a la pantalla de rutina.
	-	Se debe mantener la categoría y nivel previamente seleccionados.
	-	Puede registrar el evento “inicio de rutina” si aplica para métricas.
