**dado** que ya seleccione una rutina del listado de _"Exploración de rutinas"_

**como** usuario/socio de Planet Fitness,

**quiero** ver una recomendación sobre la duración óptima del nivel,

**para** saber cuánto tiempo se sugiere realizarlo antes de avanzar.


## Criterios de aceptación:

  -	El mensaje debe indicar que el nivel está diseñado para realizarse durante al menos 4 semanas.
	-	El texto debe estar dentro de una tarjeta visualmente diferenciada.
	-	Debe incluir una llamada a la acción para consultar a un entrenador si hay dudas.

### Mensaje recomendación

> RECOMENDACIÓN:
> Para un aprovechamiento óptimo, cada nivel
> de nuestras rutinas está diseñado para
> realizarse durante al menos 4 semanas.

### Mensaje llamada a la acción

> Si tienes alguna duda,
> consulta a un entrenador.
