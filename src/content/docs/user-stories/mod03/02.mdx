**dado** que ya seleccione una rutina del listado de _"Exploración de rutinas"_

**como** usuario/socio de Planet Fitness,

**quiero** ver un mensaje de advertencia antes de iniciar la rutina,

**para** estar consciente de que la ejecución de los ejercicios es bajo mi propia responsabilidad.

## Criterios de aceptación:

  - Debe mostrarse un aviso legal o de seguridad de forma clara y visualmente destacada.
  - Puede incluir íconos de advertencia o elementos gráficos para atraer la atención.

### Mensaje
_A lo platicado este mensaje puede cambiar, no usar hasta el Vo.Bo de Bussines._

```text
Las rutinas prediseñadas se proporcionan únicamente como sugerencia y con fines informativos.
Planet Fitness® no se hace responsable por lesiones que puedan derivarse de una ejecución incorrecta o del uso inadecuado del equipo.
Se recomienda consultar a su médico antes de comenzar cualquier rutina de ejercicio.
```

## Comentarios

Sabemos que esta decisión puede modificar el comportamiento esperado de la aplicación. A continuación, te compartimos algunas opciones que podrían implementarse:

1. Mostrar la información justo después de escanear el código QR.
2. Incluir únicamente una leyenda como **"Acepto los _Términos y Condiciones_"**, la cual debe redirigir a una nueva pantalla donde se explique de forma
    clara y textual la parte legal y las responsabilidades correspondientes.
