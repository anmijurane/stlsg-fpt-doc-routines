**dado** que ya seleccione una rutina del listado de _"Exploración de rutinas"_, lei las indicaciones y decidi continuar
**como** usuario/socio de Planet Fitness,

**quiero** que la rutina esté dividida por días,

**para** seguir un orden estructurado en mis entrenamientos.

### Criterios de aceptación:

  - Se deben mostrar tarjetas para Día 1, Día 2, Día 3 y Día 4.
  - Cada tarjeta debe tener un chevron para desplegar o colapsar sus ejercicios.
  - Cada tarjeta debe tener el titulo del ejericio a realizar.
  - Las tarjetas están colapsadas por defecto al ingresar a la vista.
  - Se debe visualizar seguido del día la parte a trabajar, si hay cardio, clase o descanso.
