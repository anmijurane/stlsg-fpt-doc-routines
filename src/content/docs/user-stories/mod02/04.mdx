**Dado** que ya escanee el código QR y estoy dentro de la plataforma

**Y** veo las categorias en listadas,

**Y** la categoria que elegi ya esta expandida visiblemente,

**Y** vi la información del nivel,

**como** usuario/socio de Planet Fitness,

**quiero** poder seleccionar un nivel dentro de una categoría,

**para** para comenzar con la rutina correspondiente.


## Criterios de aceptación

  - Al hacer clic en una tarjeta de nivel, el sistema debe redirigir al usuario a la pantalla informativa (pantalla 2).
  - El sistema debe identificar internamente la categoría y nivel seleccionados para usarlos en las siguientes pantallas.

## Alcance
  - Dentro del alcance del MVP 1, no se navegará a la pantalla informativa eso es hasta el alcance del MVP2.
