**como** plataforma,

**debo** conservar el identificador del club mientras el usuario navega,

**para** poder relacionar todos sus eventos (feedback, navegación, visitas) a ese club.

## Criterios de aceptación

  - El valor del club debe estar disponible para cualquier módulo durante la sesión.
  - El dato debe persistir incluso si el usuario navega entre pantallas.
