**como** plataforma,

**debo** detectar el club de origen mediante un parámetro en la URL,

**para** contextualizar el acceso y las métricas asociadas.

## Criterios de aceptación

  - El sistema debe leer el parámetro club desde la URL al cargar la plataforma.
  - El valor debe almacenarse en una cookie o sessionStorage para usarse durante toda la sesión.
  - Si el parametro no existe o no corresponde con el registro de alguna sede, debe redirigir al sitio web [Planet Fitness | Un gimnasio y un club de fitness para todos](https://www.planetfitness.mx/)
